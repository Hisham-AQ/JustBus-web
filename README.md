# 🖥️ JustBus Admin Dashboard – Smart Transportation Management System

JustBus Web Dashboard is the administrative and operational management platform of the JustBus ecosystem. The dashboard provides administrators with complete control over transportation operations, including trip management, driver assignment, bus monitoring, student management, emergency handling, and real-time system analytics.

The platform was developed to support efficient university transportation management through a centralized, scalable, and user-friendly web interface.

---

## 🚀 Core Features

### 👨‍💼 Administrative Features
- Full trip and route management
- Driver assignment and monitoring
- Bus management and seat capacity control
- User and student management
- Real-time transportation monitoring
- Blacklist and misconduct management
- Parcel and special trip request management
- Emergency panic alert monitoring
- Rewards and loyalty system management
- Dashboard statistics and analytics
- Live trip status monitoring

---

### 🚌 Transportation Management
- Create and manage transportation trips
- Assign buses and drivers to trips
- Manage trip schedules and statuses
- Monitor ongoing trips in real-time
- Track available seats dynamically
- View driver activity and trip history

### 🚨 Safety & Monitoring

- Panic system monitoring
